About the Universal Hardware Tester

Your ultimate tool for hardware testing, management, and automation.

About the Application

The Universal Hardware Tester is designed to simplify and optimize the testing and management of valves, spec sheets, and test plans. Follow the instructions below to unlock its full potential.

Getting Started

  1. Create or Identify Your Valve:

    Ensure the valve you want to test is listed in the application. If not, add a new valve using the Add Valve page. Provide details like the valve's name, type, optional API endpoint, and specifications.

  2. Upload Spec Sheets:

    Upload spec sheets via the Upload Spec Sheet page. This helps associate critical documentation with your valves.

    • Accepted formats: PDF, DOCX, XLSX
    • Optional: Link the spec sheet to a specific valve by providing its ID.
  3. Upload Test Plans:

    Use the Upload Test Plan page to upload your test plans. These plans will be parsed and stored for execution.

    • Accepted formats: PDF, CSV, TXT
    • Include steps like "Open Valve 50%" or "Set Pressure to 10 PSI".
  4. Run Test Plans:

    Navigate to the Run Test Plan page to execute a test plan. Simply provide the ID of the test plan to start testing.

  5. View Test Results:

    After running a test plan, view results on the View Results page. Logs and statuses for each step will be displayed for your analysis.

Additional Features

  • Real-Time Logs: Monitor live logs during test plan execution under the "Real-Time Logs" section on the dashboard.
  • Dashboard Overview: Access key features and live updates quickly from the Dashboard.
  • Search & Filter: Use search functionality to quickly locate specific valves, spec sheets, or test plans.
